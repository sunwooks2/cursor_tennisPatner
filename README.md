# 테니스 대진표 자동 생성 웹사이트 기능명세서

## 1. 프로젝트 개요

### 서비스명

테니스 대진표 자동 생성기

### 목적

테니스 동호회 운영자가 참가자 명단을 입력하지 않고,

* 남성 인원수
* 여성 인원수
* 코트 수
* 운영 시간

만 입력하면 자동으로 대진표를 생성하는 웹서비스

### 서비스 특징

* 로그인 없음
* 회원가입 없음
* 참가자 정보 입력 없음
* 모바일 최적화
* 즉시 생성
* 익명 참가자 자동 생성

예)

남1
남2
남3

여1
여2
여3

## 3.3 경기 수 계산

시스템 자동 계산

### 계산식

총가능경기수

=
코트수 × 시간슬롯수

예시

3코트

09:00 ~ 12:00

30분 경기

총 6라운드
