# 테니스 대진표 자동 생성 웹사이트 기능명세서

## 1. 프로젝트 개요

### 서비스명

테니스 대진표 자동 생성기

### 목적

테니스 동호회 운영자가 참가자 명단을 입력하지 않고,

* 남성 인원수
* 여성 인원수
* 코트 수
* 운영 시간

만 입력하면 자동으로 대진표를 생성하는 웹서비스

### 서비스 특징

* 로그인 없음
* 회원가입 없음
* 참가자 정보 입력 없음
* 모바일 최적화
* 즉시 생성
* 익명 참가자 자동 생성
* **자유 로테이션** · **팀전** 두 가지 모드

예)

남1
남2
남3

여1
여2
여3

## 2.4 팀전 모드

### 목적

동호회·클럽 간 **2개 팀 대결** 형태로 대진표를 자동 생성한다.

### 팀 입력 (팀 2개 고정)

| 항목 | 타입 | 필수 | 기본값 |
| ---- | ---- | ---- | ------ |
| 팀 A 이름 | 텍스트 | O | A팀 |
| 팀 B 이름 | 텍스트 | O | B팀 |
| 팀별 남성 인원 | 숫자 | O | 0 |
| 팀별 여성 인원 | 숫자 | O | 0 |
| 팀별 남/여 이름 | 텍스트 | X | 강남남1, 강남여1… |

이름 미입력 시 `{팀이름}남1`, `{팀이름}여1` 형식으로 자동 생성한다. (예: 강남남1, 송파여2)

### 공통 입력 (자유 로테이션과 동일)

* 코트 수
* 시작시간 / 종료시간 / 경기시간(분)
* 경기 유형 (MD / WD / MXD)

### 팀전 경기 규칙

| 유형 | 팀 A | VS | 팀 B |
| ---- | ---- | -- | ---- |
| MD | A팀 남2 | VS | B팀 남2 |
| WD | A팀 여2 | VS | B팀 여2 |
| MXD | A팀 남1+여1 | VS | B팀 남1+여1 |

### 팀전 생성 원칙

* 파트너는 **같은 팀만**
* 상대는 **다른 팀만**
* 동일 경기(4명 조합) 중복 금지
* 같은 팀 파트너·상대 팀 만남 반복 최소화
* **개인별 경기 수 균등** 우선 (팀 인원 불균형 시 가능한 범위 내 균등)
* 코트·시간 중복 금지

### 팀전 검증

| 조건 | 메시지 |
| ---- | ------ |
| 팀당 총 인원 ≥ 1 | 각 팀에 최소 1명 이상 필요 |
| MD 선택 | 팀당 남성 2명 이상 |
| WD 선택 | 팀당 여성 2명 이상 |
| MXD 선택 | 팀당 남·여 각 1명 이상 |

### 팀전 출력

* 대진표 상단 **요약**에만 팀명 표시  
  예: `강남 (남6 여4) VS 송파 (남5 여5) · 총 18경기 · 팀 출전 강남 36 / 송파 30`
* 개별 경기 셀에는 선수명만 표시 (팀명 반복 없음)
* 참가자별 페어/상대 통계 (페어=같은 팀, 상대=상대 팀)

### 팀전 공유 URL

```
?mode=team&ta=A팀&tam=6&taf=4&tb=B팀&tbm=5&tbf=5&c=3&s=09:00&e=12:00&d=30&t=MD,WD&seed=...
```

`mode` 없으면 기존 자유 로테이션으로 처리 (하위 호환).

### 팀전 보내기

* 공유 · 이미지저장 · 엑셀저장 · 인쇄 모두 팀 요약·명단 반영
